About Arena Game

Arena Game is a real-time 3D multiplayer strategy game where two players battle for supremacy using tanks, helicopters, and turrets.

This entire game was coded by Claude Code (Anthropic's AI coding assistant), with prompts and direction provided by @tombuildsstuff.

The project demonstrates what's possible when humans and AI collaborate on software development.
